## Problem

Strategic planning is fragmented across chats, documents, meetings, and dashboards. AI makes ideas faster, but most teams still lack structured critique, memory, and decision-ready synthesis.

## Workflow

1. Enter mission.
2. Select mode.
3. Agents activate.
4. Debate exposes weak assumptions.
5. Memory stores conclusions.
6. NEXUS generates the strategic directive.

## Agent Ecosystem

- Orchestrator: decomposes and coordinates.
- Researcher: maps market and problem evidence.
- Architect: designs system workflow.
- Strategist: builds GTM and business logic.
- Critic: attacks risk and assumptions.
- Memory Agent: preserves mission context.

## Strategic Cognition

NEXUS does not hide reasoning inside a single response. It makes the workflow visible: streams, graph expansion, debate, risk scoring, and session recall.

## Architecture

React + Vite frontend, FastAPI streaming backend, OpenAI Responses API-ready agent layer, lightweight local memory, and deterministic fallback for stable demos.
